Portfolio Siteplan

Purpose

This website showcases my work as a developer, including personal, academic, and collaborative projects. It serves as a platform for potential employers, collaborators, and peers to view my skills, background, and contact information.

Audience

The primary audience includes hiring managers, recruiters, tech mentors, classmates, and anyone interested in my coding journey and capabilities.

Dynamic Elements

The website will include JavaScript-powered features such as:

  • A dynamic project gallery that displays project cards filtered by technology tags using filter() and forEach().
  • A light/dark mode toggle using event listeners and DOM manipulation.
  • A project detail modal that opens on click, pulling information from an object array.
The JS code will be modularized into ES Modules and include conditional rendering based on user actions.

Logo

The logo will be a simple icon and my name (“Thomas Lucas”) styled professionally. I’ll either create it on Canva or use a developer-themed SVG.

Colors and Fonts

  • Primary Color: #003f5c (Dark Blue)
  • Accent Color: #ffa600 (Warm Yellow)
  • Background Color: #f5f5f5 (Light Gray)
  • Font Heading: 'Roboto Slab', serif
  • Font Body: 'Open Sans', sans-serif

Content

The homepage will introduce who I am as a developer, highlight 2–3 featured projects with images, and link to the child page.

The child page will contain a full list of my projects with filters, detailed descriptions, GitHub links, technologies used, and screenshots.

Both pages will include contact info and a short "About Me" section.
